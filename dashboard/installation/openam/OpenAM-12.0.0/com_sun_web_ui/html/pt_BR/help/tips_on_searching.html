Dicas sobre pesquisa

Para melhorar sua pesquisa:

  • Digite palavras ou frases prováveis dos tópicos da ajuda.
  • Não use caracteres curinga ("*", por exemplo). A pesquisa encontrará correspondências parciais sem elas: uma pesquisa por "libraries" também encontrará "library".
  • Se retornarem tópicos demais, digite mais palavras para tornar a pesquisa mais específica.
  • Se retornarem poucos tópicos, exclua ou altere as palavras no campo Pesquisa.

Nota: operadores booleanos, como AND, OR e NOT, são ignorados no campo Pesquisa.

A pesquisa compara as palavras e as frases no campo Pesquisa com as palavras e as frases nos tópicos da ajuda. Os achados são classificados com base nos seguintes critérios de correspondência de termos do campo Pesquisa e dos tópicos da ajuda:

  • todos ocorrem
  • ocorrem com o mesmo formato
  • ocorrem na mesma ordem
  • ocorrem sem palavras intervenientes
