검색 팁

검색을 보다 용이하게 하려면:

  • 도움말 항목에 있을 법한 단어 또는 절을 입력합니다.
  • 와일드카드 문자(예: "*")를 사용하지 마십시오. "libraries"에 대해 "library"도 검색하는 것과 같이 검색은 와일드카드 문자 없이 부분적으로 일치되는 단어를 찾습니다.
  • 너무 많은 항목을 나타낼 경우 - 보다 구체적인 검색어를 더 많이 입력하십시오.
  • 너무 적은 항목을 나타낼 경우 - 검색 필드의 단어를 삭제하거나 변경하십시오.

참고: 검색 필드의 및, 또는 그리고 아닌과 같은 부울 조작자는 무시됩니다.

검색은 도움말 항목의 단어 및 절과 검색 필드의 단어 및 절을 비교합니다. 검색 결과는 검색 필드 및 도움말 항목에 있는 순으로 다음과 같이 나열됩니다.

  • 모두 발생
  • 같은 형태로 발생
  • 같은 순서로 발생
  • 단어간 조정 없이 발생
