Suggerimenti sulla ricerca

Per migliorare l'efficacia delle ricerche:

  • Digitare parole o frasi che hanno una buona probabilità di essere contenute negli argomenti della Guida.
  • Non usare i metacaratteri (ad esempio "*"). La procedura di ricerca è in grado di trovare le corrispondenze parziali anche senza l'uso dei metacaratteri: la ricerca di "librerie" trova anche "libreria".
  • Se vengono restituiti troppi argomenti, inserire altre parole per rendere più specifica la ricerca.
  • Se vengono restituiti pochi argomenti, eliminare o modificare le parole nel campo Ricerca.

Nota: gli operatori logici, come AND, OR e NOT sono ignorati nel campo Ricerca.

La ricerca confronta le parole e le frasi presenti nel campo Ricerca con quelle contenute negli argomenti della Guida. I risultati vengono ordinati in base al fatto che negli argomenti della Guida i termini presenti nel campo Ricerca:

  • sono tutti presenti
  • si presentano nella stessa forma
  • si presentano nello stesso ordine
  • si presentano senza parole interposte
