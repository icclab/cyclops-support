搜索技巧

以下技巧可帮助您提高搜索效率：

  • 键入可能出现在帮助主题中的单词或短语。
  • 不要使用通配符（如 "*"）。不使用通配符会得到部分匹配的搜索结果： 即搜索 "libraries" 时也将搜索到 "library"。
  • 如果搜索返回的主题过多 - 请输入更多的单词进一步限定搜索范围。
  • 如果搜索返回的主题过少 - 请删除或更改“搜索”字段中的单词。

注意：布尔运算符（如 AND、OR 和 NOT）在“搜索”字段中将被忽略。

搜索过程将“搜索”字段中的单词和短语与帮助主题中的单词和短语进行比较。搜索结果根据“搜索”字段和帮助主题中的术语是否满足以下情形进行排序：

  • 全部出现
  • 以相同的形式出现
  • 以相同的顺序出现
  • 之间没有出现其他单词
