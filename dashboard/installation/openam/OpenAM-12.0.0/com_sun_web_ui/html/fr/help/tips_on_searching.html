Conseils sur la recherche

Optimisation de vos recherches

  • Tapez des mots ou des expressions qui ont de fortes chances de figurer dans les rubriques d'aide.
  • N'utilisez pas de caractères génériques (tels que "*"). Le moteur de recherche trouve des occurrences partielles sans cet opérateur : la recherche du terme "bibliothèques" permettra également de trouver "bibliothèque".
  • Si vous obtenez trop de résultats, saisissez davantage de mots afin d'affiner la recherche.
  • Si vous n'obtenez pas assez de résultats, supprimez ou modifiez les mots dans le champ de la recherche.

Remarque : les opérateurs booléens, tels que ET, OU et NON, ne sont pas pris en compte dans les recherches.

La fonction de recherche compare les mots et expressions tapés dans le champ de la recherche à ceux qui figurent dans les rubriques d'aide. Les occurrences trouvées sont classées dans le champ de la recherche et les rubriques d'aide selon les critères suivants :

  • Tous les termes apparaissent.
  • Tous les termes apparaissent au même format.
  • Tous les termes apparaissent dans le même ordre.
  • Tous les termes apparaissent sans mot tiers placé entre eux.
