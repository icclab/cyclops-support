搜尋提示

若要讓您的搜尋更為有效：

  • 鍵入可能會出現於說明主題的字句。
  • 請勿使用萬用字元（例如：「*」）。搜尋可找出不含萬用字元的部分相符結果：例如在搜尋「libraries」時也會找到「library」。
  • 如果搜尋結果出現太多主題 － 請輸入更多字，讓搜尋更為特定準確。
  • 如果搜尋結果出現太少主題 － 請刪除或變更 [搜尋] 欄位中的字。

注意：「搜尋」欄位中的布林運算符號（例如：AND、OR 和 NOT）會被忽略。

搜尋會比較 [搜尋] 欄位中的字句和說明主題中的字句。搜尋的分類是根據 [搜尋] 欄位和說明主題中的詞語是否：

  • 都有出現
  • 以相同的形式出現
  • 以相同的順序出現
  • 出現時中間未插入其他字詞
